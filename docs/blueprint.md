# **App Name**: TrackFlow

## Core Features:

- Inventory Tracking: Add, edit, and track items in the inventory with details like original value and selling value.
- Sales Recording: Record sales with the actual selling price after bargaining, capture transaction details, and associate them with inventory items.
- Earnings Calculation: Calculate daily, monthly, and annual earnings, revenue, and profit based on sales and inventory data.
- Stock Management: Track the quantity of items added to the inventory and the quantity of items sold.
- Report Generation: Create custom reports to analyze sales trends, best-selling products, and overall financial performance.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) to create a calm and trustworthy atmosphere, reflecting the app's function as a reliable tracking tool.
- Background color: Very Light Blue (#F0F8FF) to keep the interface clean and allow the information to stand out.
- Accent color: Light Purple (#CDB4DB) as it is analogous to soft blue; slightly desaturated, this is for interactive elements and to maintain visual interest.
- Font: 'Inter' sans-serif for both headings and body text, for a clean and easily readable minimalistic design.
- Simple line icons to represent inventory items and sales data.
- Minimalist layout with clear sections for inventory, sales, and reports.
- Subtle transitions and animations to provide feedback on interactions without being distracting.
